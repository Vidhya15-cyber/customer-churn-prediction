# 📉 Customer Churn Prediction

## 📌 Overview

Customer Churn Prediction is a Machine Learning project that predicts whether a customer is likely to leave a company based on customer behavior and historical data. The model helps businesses improve customer retention by identifying at-risk customers early.

---

## ✨ Features

- 📊 Customer data analysis
- 🤖 Machine Learning-based prediction
- 📈 Customer churn classification
- 📉 Data preprocessing and feature engineering
- 💻 Interactive prediction interface
- ⚡ Fast and accurate predictions

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/customer-churn-prediction.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```

or

```bash
python app.py
```
